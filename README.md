# Financial RAG Lab

This project builds a Retrieval-Augmented Generation system using the Databricks OfficeQA Treasury Bulletin dataset.

## Goal

Compare a baseline RAG system with an engineered RAG system using retrieval and answer quality metrics.

## Years Used

2022, 2023, 2024, 2025

## Metrics

- Hit Rate@5
- MRR
- Groundedness
- Factual Accuracy
- Hallucination Rate
